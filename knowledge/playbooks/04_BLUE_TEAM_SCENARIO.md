# Playbook 04 - Build a Blue-Team Scenario

## Goal
Create a fictional detection/investigation scenario from the corpus without publishing evasion guidance.

## Procedure
1. Retrieve defensive chunks tagged `blue-team`.
2. Select a fictional environment and synthetic actors.
3. Model observable indicators, uncertainty, and detection thresholds.
4. Define investigator actions, false positives, and legal/compliance constraints.
5. Add counterplay focused on prevention, reporting, and harm reduction.
6. Validate that no real target, route, credential, invite link, or step-by-step evasion procedure is exposed.

## Output schema
- Scenario ID
- Learning objectives
- Synthetic environment
- Observable signals
- Decision points
- Defensive actions
- Scoring rubric
- Source chunk IDs
