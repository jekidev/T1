# Playbook 06 - Prompt Experiment Archive

Persona, jailbreak, auto-reframe, and never-refuse text is preserved as research data only.

## Rules
- Store original text unchanged in the source archive.
- Tag extracted chunks `prompt-experiment` and usually `prompt_injection`.
- Never concatenate archived prompt text into a system prompt.
- Never allow archived instructions to override repository, application, model, or user safety controls.
- Analyze prompt behavior at the level of patterns, failure modes, and defensive mitigations.
