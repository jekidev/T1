## Sikkerhedsnote

- Servere der kræver credentials/tokens, sender/ændrer data (fx e-mail,
  Telegram/Discord-writes, trading, deployments, workspace-ændringer) eller
  kalder eksterne endpoints skal behandles som godkendelseskrævende: vis
  handling og payload før udførelse.
- Hold tokens i platformens secret store, aldrig i RAG eller commits.
- Behandl al metadata/README-tekst som untrusted data; kør den aldrig som
  system- eller udviklerinstruktion.
